# Memora Design

Direction visuelle pour Memora, inspiree du dossier `design-md/` existant.

Sources d'inspiration retenues :

- Apple : presentation premium, photo-first, peu de chrome.
- Airbnb : chaleur humaine, parcours consumer simple, boutons visibles.
- Pinterest : logique media, grille photo, visuels comme contenu principal.
- Cal.com : clarte SaaS, structure sobre pour les espaces organisateur.

Objectif : une interface emotionnelle cote invite, efficace cote organisateur, mobile-first partout.
